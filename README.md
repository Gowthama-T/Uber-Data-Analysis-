# 🚖 Uber Ride Booking Analysis Dashboard | Tableau

## 📌 Project Overview

The **Uber Ride Booking Analysis Dashboard** is an interactive Tableau dashboard developed to analyze Uber ride booking data. The project provides insights into booking trends, revenue generation, customer behavior, ride patterns, and operational performance through dynamic visualizations and filters.

The dashboard is divided into two sections:
- **Summary Dashboard** – High-level business overview
- **Detailed Dashboard** – Deep dive into operational insights

---

## 🎯 Objectives

- Analyze Uber ride booking performance.
- Monitor booking trends over time.
- Evaluate revenue generation.
- Identify top-performing pickup locations.
- Understand customer ride behavior.
- Support business decision-making using interactive dashboards.

---

## 📂 Dataset Information

The dataset contains ride booking records with the following attributes:

- Date
- Time
- Booking ID
- Booking Status
- Customer ID
- Vehicle Type
- Pickup Location
- Drop Location
- Avg VTAT (Vehicle Time to Arrival)
- Avg CTAT (Customer Time to Arrival)
- Cancelled Rides by Customer
- Customer Cancellation Reason
- Cancelled Rides by Driver
- Driver Cancellation Reason
- Incomplete Rides
- Incomplete Ride Reason
- Booking Value
- Ride Distance
- Driver Rating
- Customer Rating
- Payment Method
